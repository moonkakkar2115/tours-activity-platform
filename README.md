## Run the project (DB + DevOps setup)

### Prerequisites
- Docker
- Docker Compose

### Steps to run

```bash
git clone https://github.com/moonkakkar2115/tours-activity-platform.git
cd tours-activity-platform
docker compose up --build
```
### Services

- Frontend: http://localhost:3000
- Backend: http://localhost:5000
- MongoDB: mongodb://localhost:27017
